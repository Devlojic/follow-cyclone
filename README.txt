Cette application web a pour but d’aider les passions de météorologie de suivre les systèmes dépressionnaires naissant dans les zones tropicales.

Details: 

L’objectif est qu’un maximum d’informations se trouve sur la carte interactive. Cet outil dynamique permettra à l’utilisateur de calculer des distances, mais aussi faire ses propres prévisions.

L’application web popover également son propre système d’alerte automatique qui avertira les abonnés à l’approche d’un danger
